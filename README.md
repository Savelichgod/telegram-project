# telegram-project

Для запуска нужно создать активировать виртуальное окружение командами

```
python -m venv venv
source venv/Scripts/activate
```
Перед запуском установите requirements командой

```
pip install -r requirements.txt
```
Для запуска
```
python bot.py
```
